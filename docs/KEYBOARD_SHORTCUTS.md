# Keyboard Shortcuts

- `Ctrl+Shift+Space`: Open Glyph
- `Tab` / `Shift+Tab`: Move focus between search, history, preview actions, and settings
- `Up` / `Down`: Navigate history results
- `Enter`: Run the default action for the selected item
- `CmdOrCtrl+F`: Focus search
- `CmdOrCtrl+Backspace`: Delete selected item
- `CmdOrCtrl+L`: Clear search
- `CmdOrCtrl+.`: Open settings
- `CmdOrCtrl+Shift+F`: Toggle favorite
- `Escape`: Close settings or hide the window
